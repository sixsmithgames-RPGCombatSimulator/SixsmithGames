## Overview
MasterTyping is a practical typing trainer designed for people who spend real time on a keyboard. It combines quick baselines, focused drills, repeatable practice sessions, and a lighter game mode so users can improve speed, accuracy, and consistency without the product feeling like classroom software.

## Modes
- **Game mode**: Playful stages with themed prompts and light progression.
- **Pro mode**: Skill drills for speed and accuracy with stricter scoring.
- **Assessment mode**: Six-step assessment that pinpoints weaknesses (home row, weak fingers, shift keys, common patterns, speed) with upgradeable deep analytics for detailed breakdowns.
- **Exercise / specialized practice**: Targeted drills by skill focus to address weaknesses identified in assessments.

## Key Features
- **Quick baseline flow**: First-run onboarding guides users into a fast assessment or a lighter game-first start based on intent.
- **Focused improvement loop**: Assessments feed directly into drills or skill-focused Pro Mode sessions so users know what to practice next.
- **Assessment analytics**: Comprehensive six-exercise assessment with basic results for all users and optional deep analytics (per-exercise breakdowns, weakness detection, personalized recommendations).
- **Recent progress tracking**: Per-mode stats (game, pro, assessment, exercises), WPM trend chart, recent sessions, and recurring weak-spot tracking—all saved locally in the browser.
- **Rolling retention model**: Recent session history is kept for up to 90 days so the dashboard stays current and useful instead of turning into a permanent archive.
- **Engaging practice options**: Game Mode keeps repetition fun, while Pro Mode and drills provide repeatable training for users who want a more direct improvement path.
- **Low-friction premium**: Core training is free; premium focuses on deeper insights rather than locking the main practice loop.

## Monetization
- **Free**: Full access to Game Mode, Pro Mode, assessments with core results, focused drills, and recent progress tracking.
- **Premium**: Unlock deeper assessment insights, richer weakness detection, and more detailed follow-up guidance.

## Audience Notes
- Creators, writers, and note-takers who want cleaner keyboard flow.
- Gamers, streamers, and chat-heavy users who want speed without sloppy errors.
- Students and knowledge workers who need a practical way to improve baseline typing and keep progress visible.
- Anyone who wants a useful free typing trainer first, with premium reserved for deeper insight rather than locked core practice.
